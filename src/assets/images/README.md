# Logo图片说明

## 文件位置
请将logo图片文件命名为 `logo.png` 并放置在此目录中。

## 文件要求
- **文件名**: `logo.png` 
- **格式**: PNG (推荐，支持透明背景) 或 JPG
- **尺寸**: 建议高度 120-200px，宽度自适应
- **质量**: 高清图片，确保在不同设备上显示清晰

## 支持的格式
- PNG (推荐)
- JPG/JPEG
- SVG
- WebP

## 使用说明
1. 将logo图片文件复制到 `src/assets/images/` 目录
2. 确保文件名为 `logo.png`
3. 重新构建项目：`npm run build`
4. logo将自动显示在网站左上角

## 注意事项
- 如果图片加载失败，会自动显示文字版本的logo
- 图片会自动适应导航栏高度
- 支持高DPI屏幕显示
